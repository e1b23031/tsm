# Oni Soohting 

Xboxのコントローラーを用いて行うシューティングゲーム。 

 

### UnityHubのダウンロード 

1. https://unity.com/ja/download からUnityhubをインストールする。 

2. UnityHubのInstallsタブのInstallEditerから6000.2.7f2を選択する。

3. もしくはUnityhub 6000.2.7f2と検索する 

 

### クローン 

1.　以下のURLからGitHubにアクセスする。 

　　https://github.com/e1b23031/tsm

2.　Code　→　Download ZIP 

3.　ダウンロードしたファイルを「すべて展開」する。 

　　（※展開するファイルはUnityHubを選択してください。） 

 

### 起動 

1.　UnityHubを起動し、「Add」をクリックし、続けて「Add project from disc」をクリック。 

2.　先ほど解凍したファイルがあるUnityHubを選択する。 

3.　Projectを開く。 

4.　左下のScenesをクリックしてその横にあるStartをクリック。 

5.　画面右側にあるStart（Scene Asset）を「Open」する。 

6．左上の「Game」を右クリックし、「Maximize」を選択 

7.　playボタンをクリックする。 

 

### コントローラーの接続 

1．有線のXboxコントローラーをパソコンのポートに指し接続する。 

2．中央のボタンを長押しする。ボタン下が光れば完了。 

 

##Start画面 

1．EnterKeyをクリックしmain画面に進む 

 

##game画面 

1．照準移動 

　　→コントローラーの左スティックと連動し、上下左右の操作ができる 

2．発射 

　　→背面のLT,RTで射撃。押し込みでは出ない。連打が必要。 

 

##Finish画面 

1．画面右下の「Retry」ボタンでStart画面に遷移 

2．画面左下の「Quit」ボタンでゲーム終了
